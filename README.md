# ThreadFlow

2つのThreadsアカウントを、無料テンプレート・Google Sheets・Google Apps Script・Renderの投稿ブリッジで運用する管理画面です。

## 無料自動投稿の流れ

1. Google Sheetsの「アカウント設定」「投稿プロフィール」を読み取る
2. Apps Scriptが15分ごとに管理画面へ同期する
3. 日本時間の投稿枠と投稿比率に合わせて、追加料金なしのテンプレートで投稿文を作る
4. 1アカウント1日10件を上限にThreadsへ投稿する
5. 生成内容と結果を「投稿キュー」「投稿ネタ」へ戻す

自動投稿がOFFのアカウントは生成も投稿もしません。「確認モード」がONの場合は、Threadsへ投稿せず管理画面の確認待ちへ保存します。

## Apps Script

`gas/Code.gs` を対象スプレッドシートに紐づくApps Scriptへ貼り付けます。スプレッドシートを再読み込みすると「ThreadFlow」メニューが表示されます。

1. 「接続設定」で管理画面URLと自動実行キーを保存
2. 「今すぐ同期・実行」で接続テスト
3. 「15分ごとの自動実行を開始」で無料トリガーを登録

秘密のキーはスプレッドシートのセルやGitHubには保存されず、Apps Scriptのスクリプトプロパティに保存されます。

## 開発

```bash
npm install
npm run build
```

Node.js 22.13以上を使用します。
